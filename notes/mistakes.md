# Mistakes

## Main mistake

Trying to make things "cool" instead of making them "work".

## Technical mistakes

* Too many layers (overlays, animations)
* No control over z-index
* Event handling issues
* No debugging step

## Thinking mistakes

* Rushing development
* Skipping fundamentals
* Trusting AI too much

## Lesson

Good software is:

* simple
* understandable
* stable

Not complex.
