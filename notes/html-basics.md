# HTML Basics

## What is HTML?
HTML is used to structure web pages.

---

## Basic Structure

- html → root of the page
- head → invisible (title, meta)
- body → visible content

---

## Common Tags

### Headings
- h1 → main heading
- h2 → subheading

### Text
- p → paragraph

### Lists
- ul → unordered list
- li → list item

### Links
- a → hyperlink

### Images
- img → display image

---

## Important Notes

- Always use proper structure (html, head, body)
- Avoid using only divs
- Write clean and readable code

---

## Example Thinking

- Title → h1
- Description → p
- Items → ul/li
