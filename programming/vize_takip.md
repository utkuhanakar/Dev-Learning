# Python Vize Hazırlık Takip Listesi (Issue Tracker)

Bu dosya, vize sınavına kadar tamamlamanız gereken konuları birer "Issue" gibi takip etmeniz için oluşturulmuştur.

## 📝 Tamamlanması Gereken Görevler

- [ ] **Değişkenler ve Temel Veri Tipleri**
  - [x] Değişken atamaları (x = 5)
  - [ ] Tür dönüşümleri (int, float, str)
  - [ ] PEP8 kurallarına uyum

- [ ] **Karar Yapıları ve Mantıksal Dallanma**
  - [ ] if-elif-else blokları
  - [ ] Mantıksal operatörler (and, or, not)
  - [ ] Boolean değerli ifadeler

- [ ] **Fonksiyonlar ve Kapsam (Scope)**
  - [ ] Fonksiyon tanımlama ve çağırma
  - [ ] Yerel (local) ve Global değişken farkı
  - [ ] Docstring ve Doctest kullanımı
  - [ ] Birden fazla değer döndürme (return a, b)
  - [ ] Lambda fonksiyonları

- [ ] **Argüman Yönetimi**
  - [ ] Belirli sayıda argümanlar
  - [ ] Belirsiz sayıda argüman kullanımı (*args)
  - [ ] İsimli belirsiz argümanlar (**kwargs)

- [ ] **Veri Yapıları: Listeler ve Stringler**
  - [ ] Liste tanımlama ve temel metodlar (append, pop)
  - [ ] String manipülasyonu ve parçalama (split, strip)
  - [ ] Liste dilimleme (Slicing)

- [ ] **Döngüler ve Yineleme**
  - [ ] for ve while döngüleri
  - [ ] range() fonksiyonu

- [ ] **Hata Ayıklama (Debug & Exception Handling)**
  - [ ] try-except blokları
  - [ ] raise ile hata fırlatma
  - [ ] Mantıksal hataları bulma
