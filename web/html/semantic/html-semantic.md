# HTML Semantic

## What is Semantic HTML?

Semantic HTML means using meaningful tags.

---

## Common Semantic Tags

- header → top section
- section → content area
- footer → bottom
- nav → navigation
- figure → image container

---

## Why Important?

- Better SEO
- Readable code
- Professional structure

---

## Example Structure

- header → name
- section → content
- footer → links
