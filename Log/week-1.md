# Week 1 Log

## Focus

Building a portfolio and learning web fundamentals.

## What happened

I tried to improve my portfolio by adding:

* entry systems
* animations
* overlays
* theme switching

At first it looked interesting, but quickly turned into a mess.

## Problems I faced

* Buttons stopped working
* Click events were blocked
* UI layers conflicted with each other
* Too many systems interacting at once
* Lost the original clean design

## Mistakes

* I added features without stabilizing the base
* I trusted AI outputs without fully understanding them
* I didn't test step by step

## What I learned

* Simplicity > complexity
* A working UI is better than a fancy broken one
* Build → test → then improve
* Every system must have a clear purpose

## Next step

Rebuild the portfolio with:

* clean structure
* minimal features
* solid interactions
